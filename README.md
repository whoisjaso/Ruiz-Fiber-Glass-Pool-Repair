# Ruiz Fiberglass Pool Repair

Static single-page lead-generation website for Ruiz Fiberglass Pool Repair in Houston, TX.

## Files

- `index.html` - page structure, SEO metadata, and local business schema
- `styles.css` - responsive brand styling
- `script.js` - mobile menu, reveal motion, before/after toggle, and form success state
- `assets/` - SVG logo, favicon, and service/gallery placeholders
- `PRODUCT.md` / `DESIGN.md` - product and visual direction for future edits

## Local Preview

Open `index.html` directly in a browser.

## Deploy

This site is static and deploys directly to Vercel from the repository root.
